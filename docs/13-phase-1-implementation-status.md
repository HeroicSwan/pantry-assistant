# Prompt 2 foundation status

The foundation runs directly on Windows with native PostgreSQL 18, Drizzle, Better Auth, server-only authorization, reviewed migrations, fictional seeds, and an isolated test database.

Implemented behavior includes sign-up/sign-in/sign-out, password-reset architecture, profiles, organizations, locations, memberships, scoped roles and permissions, invitations, onboarding, active scope, protected navigation, settings, location/team management, suspension denial, final-administrator protection, organization/location isolation, and immutable audit logs.

Source checks, clean test-database migration/seed, native integration tests, desktop/mobile Playwright tests, and a production build are part of the acceptance workflow. Later inventory and communications phases remain deferred.
